# 2단계: 맥락연결 → 영향분석

## 개요
수집된 데이터를 맥락에 연결하고 영향력을 분석합니다.

## 분석 프레임워크

### 가치 관점
- 이 변화가 **고객가치**에 어떤 영향을 미치는가?
- 기업이 변화에 **준비**되어 있는가?

## 분석 단계

### 1. 투자자심리 분석
- 현재 시장 심리 파악
- 고객별 가능행동 유형 분류

### 2. 맥락 연결
- 과거 사례와의 연관성
- 산업/섹터 간 연결고리

### 3. 영향력 평가
- 단기/중기/장기 영향 구분
- 직접/간접 영향 분류
