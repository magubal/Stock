# 1단계: 데이터 수집 (자동화)

## 개요
시장 관련 정보를 자동으로 수집하는 워크플로우입니다.

## 수집 대상
- **이슈**: 블로그, 텔레그램, 뉴스, 보고서
- **행동(시세)**: 마구별 + 현황
- **블로그 및 유튜브**: 광고, 의도 파악
- **트렌드**: 의도와 소음, 실질 방향학습

## 실행 절차

### 1. 뉴스 수집
```bash
# KITA 뉴스, 네이버 금융 등에서 RSS/크롤링
```

### 2. 텔레그램 모니터링
- 주요 투자 채널 모니터링
- 키워드 알림 설정

### 3. 증권사 보고서 파싱
- PDF/HTML 보고서 수집
- 핵심 내용 추출
