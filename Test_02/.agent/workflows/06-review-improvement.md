# 6단계: 결과확인, 복기 및 개선점 정리

## 개요
투자 결과를 분석하고 교훈을 도출하여 다음 사이클에 적용합니다.

## 복기 프레임워크

### 1. 결과 기록
- 진입가 / 청산가
- 수익률 / 손익금액
- 보유 기간

### 2. 의사결정 평가
| 항목 | 당시 판단 | 실제 결과 | 평가 |
|------|----------|----------|------|
| 시장 방향 | | | ⭕/❌ |
| 섹터 선택 | | | ⭕/❌ |
| 종목 선택 | | | ⭕/❌ |
| 타이밍 | | | ⭕/❌ |
| 비중 | | | ⭕/❌ |

### 3. 개선점 도출
- **잘한 점**: 유지/강화할 부분
- **아쉬운 점**: 개선이 필요한 부분
- **배운 점**: 새로운 인사이트
