# 5단계: 실질확인 (시세, 실제행동)

## 개요
시나리오 실행 시점에서 실시간 시세와 실제 시장 상황을 확인합니다.

## 확인 항목

### 1. 시세 확인
- 현재가 / 등락률
- 거래량 / 거래대금
- 호가 스프레드

### 2. 시장 상황
- 지수 동향
- 섹터 동향
- 관련주 동향

### 3. 실제 행동 트래킹
- 주문 실행 여부
- 체결 가격 / 수량
- 슬리피지 확인

## 실행 체크리스트

- [ ] 시나리오 조건 충족 여부 확인
- [ ] 예상과 실제 시세 비교
- [ ] 주문 실행 / 대기 결정
- [ ] 실행 결과 기록
