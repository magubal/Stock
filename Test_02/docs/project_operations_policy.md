## 🤖 **AI 어시스턴트 행동 지침**

1. **아침**: 전일 요약 → 오늘 계획
2. **중간**: 진행 상황 상세 보고
3. **저녁**: 완료 요약 + 내일 계획
4. **요청**: 사용자 검증 대기 및 피드백

### **교차 확인 포인트**
- 기능 구현 vs 초기 요청
- 품질 수준 vs 실제 사용성
- 진행 속도 vs 예상 기간
- 사용자 만족도 vs 개발 방향

### **지속 개선**
- 반복 작업 자동화
- 코드 품질 향상
- 사용자 피드백 적극 반영
