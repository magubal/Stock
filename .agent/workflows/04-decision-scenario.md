# 4단계: 의사결정 시나리오 정리

## 개요
중요도 평가를 바탕으로 구체적인 행동 시나리오를 수립합니다.

## 시나리오 템플릿

### 시나리오 A: [상황명]
- **조건**: 어떤 상황이 발생하면
- **행동**: 어떤 행동을 취할 것인가
- **목표가**: 목표 가격 설정
- **손절가**: 리스크 관리 기준
- **비중**: 포지션 크기

### 시나리오 B: [대안]
- **조건**: 다른 상황이 발생하면
- **행동**: 대안 행동
- ...

### 시나리오 S: 매도 검토 (📉)
> 참조: [investment-philosophy.md](../../docs/investment-philosophy.md)

**트리거 조건:**
- [ ] 상승 후 가격부담 발생
- [ ] 시장 변동성 심화

**평가 체크리스트:**
- [ ] 중요 이슈 데이터 수집/모니터링 완료
- [ ] 투자심리 맥락적 연계 추정 완료
- [ ] 하락 방향성 지속 여부 확인

**매도 결정:**
- 하락 방향성 **지속적 상승** → 매도 실행
- 일시적/반복적 → 모니터링 유지

## 의사결정 체크리스트

- [ ] 변화는 기회인가, 리스크인가?
- [ ] 이 기업은 변화에 준비되어 있는가?
- [ ] 고객가치혁신에 대한 준비가 되어 있는가?
- [ ] 시대의 방향(WHY)을 체감하고 있는가?

## 핵심 원칙
> "변화는 기회(매수자)이자 리스크(기존 추세종목 보유자)이고,
> 새로운 추세가 수익이자 손실이다."
