# 분석 스킬

## 개요
수집된 데이터를 분석하여 인사이트를 도출하는 스킬입니다.

## 분석 영역

### 1. 센티먼트 분석
- 시장 심리 파악
- 공포/탐욕 지수 모니터링
- 소셜 미디어 감정 분석

### 2. 영향력 평가
- 뉴스/이슈의 시장 영향력 측정
- 단기/중기/장기 영향 분리
- 섹터별 영향도 분류

### 3. 맥락 연결
- 과거 유사 사례 검색
- 패턴 인식
- 인과관계 분석

## 분석 프레임워크

```
[이슈] → [영향받는 주체] → [예상 행동] → [시장 영향]
```
