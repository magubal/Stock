# 리서치 스킬

## 개요
장기적 관점의 리서치와 트렌드 분석을 수행하는 스킬입니다.

## 리서치 영역

### 1. 기업 가치 분석
- 고객가치제안 분석
- 경쟁우위 평가
- 성장성/수익성/안정성 분석

### 2. 트렌드 리서치
탐색 영역:
- **탐색(on)**: 새로운 흐름 발견
- **탐사(off)**: 심층 분석
- **탐구**: 본질 파악

### 3. 인사이트 정리
- 독서 노트 정리
- 시장 레터 요약
- 전문가 의견 종합

## 위대한 기업 발굴 기준

> 참조: [investment-philosophy.md](../../docs/investment-philosophy.md)

### 고객가치제안 평가
```markdown
- [ ] 핵심/잠재 고객에게 경쟁사 대비 차별적 혜택이 있는가?
- [ ] 차별화 철학/미션이 명확한가?
- [ ] 실질적 행동(실행력)이 있는가?
- [ ] 경쟁력 상승의 증거가 있는가?
```

### 산업 평가 (빅트렌드)
```markdown
- [ ] 빅트렌드에 부합하는 산업인가?
- [ ] 과점/독점적 해자요인이 있는가?
- [ ] 성장 변수가 적은가? (보유리스크↓)
```

### 종합 평가
```markdown
- [ ] 변화에 민감하게 반응하는가?
- [ ] 환경을 지배하거나 바꾸는 기업인가?
```
