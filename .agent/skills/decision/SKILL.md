# 의사결정 스킬

## 개요
투자 의사결정을 지원하는 스킬입니다.

## 핵심 기능

### 1. 시나리오 작성
- 다중 시나리오 수립
- 조건부 행동 계획
- 목표가/손절가 설정

### 2. 리스크 평가
> "변화는 기회(매수자)이자 리스크(보유자)이다"

- 변화 = 기회 vs 리스크 판단
- 포지션 사이즈 결정
- 헤지 전략 수립

### 3. 기회 발굴
- 변화에 민감하게 반응하는 기업 발굴
- 고객가치혁신 준비도 평가
- 시대 방향성 적합도 검토

## 의사결정 체크리스트

```markdown
- [ ] 이 기업은 고객가치제안을 명확히 하는가?
- [ ] 변화를 기회로 전환할 준비가 되어 있는가?
- [ ] 시대의 방향(WHY)을 이해하고 있는가?
- [ ] 준비된(HOW) 기업인가?
```
