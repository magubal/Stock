# 데이터 수집 스킬

## 개요
시장 관련 데이터를 다양한 소스에서 수집하는 스킬입니다.

## 수집 소스

### 뉴스 소스
| 소스 | URL | 수집 방법 |
|------|-----|----------|
| KITA | kita.net | RSS/크롤링 |
| 네이버 금융 | finance.naver.com | RSS |
| Google News | news.google.com | RSS |

### 텔레그램 채널
- 주요 투자 정보 채널 모니터링
- 키워드 기반 알림

### 증권사 보고서
- PDF 보고서 파싱
- 핵심 내용 자동 추출

## 사용법

```
/01-data-collection
```

위 명령어로 데이터 수집 워크플로우를 실행합니다.
